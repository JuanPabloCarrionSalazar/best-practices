 ### Descripción

Utilice notaciones CSS abreviadas para reducir el tamaño de las hojas de estilo.

 ### Ejemplos

 No escriba :
 ```css
 margen superior: 1em; margen derecho: 0; margen inferior: 2em; margen izquierdo: 05 em;
 ```
Escriba en su lugar :
 ```css
 margen:1em 0 2em 05em;
 ```
